# SkillTrack 🚀

SkillTrack is a roadmap-based learning tracker that helps users stay consistent with structured daily plans and track their progress with streaks.

## 🔥 Features

- Multiple career roadmaps (Full Stack, DevOps, AI/ML, etc.)
- 7-day structured learning plan
- Task-based progress tracking
- Daily streak system 🔥
- Local storage support (data persists after refresh)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🎯 How It Works

1. Select a career path
2. View roadmap phases
3. Start learning with daily plan
4. Complete tasks
5. Track your streak and progress
